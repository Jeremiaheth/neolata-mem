Memory infrastructure for autonomous systems

Give your AI agents durable, trustworthy memory.

A production-ready, graph-native memory engine with semantic + keyword retrieval, trust-gated belief updates, and explainability built in.

Get started View npm package

Graph-native by default

Every memory can auto-link to related memories, enabling multi-hop traversal, cluster discovery, and stronger context generation for RAG workflows.

Trust-aware memory evolution

Contradictions are resolved with provenance, confidence, and trust gating. Low-trust conflicts can be quarantined for review instead of silently replacing facts.

Explainability and control

Inspect why a memory was retrieved, filtered, superseded, or quarantined. Explanations make debugging and governance straightforward.

Install in one line

npm install @jeremiaheth/neolata-mem

Why teams choose neolata-mem

  • Zero-config local JSON storage for rapid prototyping.
  • OpenAI-compatible embeddings support (OpenAI, NVIDIA NIM, Ollama, and more).
  • Built-in graph APIs: path, traversal, clusters, orphans, and health.
  • Predicate schema registry for conflict and deduplication policies.
